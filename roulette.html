The US Roulette Wheel

A US (American) roulette wheel has 38 pockets:

  • 18 red numbers
  • 18 black numbers
  • 2 green pockets: 0 and 00

The two green pockets give the house its edge. European roulette has only one green pocket (0), making it statistically more favorable for the player.

Basic Probability Formula

The probability of any event is:

P(event) = favorable outcomes / total outcomes

For a single number in US roulette:

P(single number) = 1/38 ≈ 0.02632 ≈ 2.63%

For red/black:

P(red) = 18/38 ≈ 0.4737 ≈ 47.37%

Notice it's not 50% — the two green pockets shift the odds toward the house.

House Edge Explained

The house edge is the statistical advantage built into every bet:

House Edge = (2/38) × 100% = 5.26%

This means for every $100 wagered on average, the casino expects to keep $5.26 over the long run. This is true for every bet on the table (except the top-line bet, which is worse at 7.89%).

Expected Value (EV)

Expected Value tells you the average outcome per bet over many trials.

EV = Σ (outcome × probability)

For a $1 bet on a single number (pays 35:1):

EV = (35 × 1/38) + (-1 × 37/38) EV = 0.9211 - 0.9737 EV = -$0.0526

You lose ~5.3 cents per dollar bet on average. This holds for every standard bet type.

The Law of Large Numbers

As the number of trials increases, the observed frequency of an event converges to its theoretical probability.

lim(n→∞) observed_freq/n = P(event)

After 10 spins, red might appear 70% of the time. After 10,000 spins, it will be very close to 47.37%. This is why the Gambler's Fallacy is wrong — each spin is independent!

Variance & Standard Deviation

While EV tells you the average, variance measures the spread of outcomes.

Variance = E[X²] - (E[X])² Std Dev = √Variance

For a $1 red/black bet over n spins:

σ = √(n × p × (1 - p)) σ = √(n × 0.4737 × 0.5263)

High variance means wild short-term swings even when the long-term average is known. This is why people win sometimes — and why they always lose in the long run.

The Gambler's Ruin

Even with a small house edge, a player with finite money playing against a casino with (effectively) infinite money will always go broke eventually.

P(ruin) → 1 as time → ∞ (when house has edge)

The probability of ruin increases with the number of bets placed and the size of the house edge. This is the mathematical certainty behind why casinos are profitable businesses.

1. Basic Roulette Simulation

Start with the fundamentals — modeling a roulette wheel in Python using random. This teaches random number generation, lists, and conditional logic.

roulette_basics.py 
import random

# US Roulette: 0, 00, 1-36
# We represent 00 as the string "00" and everything else as ints
pockets = [0, "00"] + list(range(1, 37))

red_numbers = {1,3,5,7,9,12,14,16,18,19,21,23,25,27,30,32,34,36}
black_numbers = {2,4,6,8,10,11,13,15,17,20,22,24,26,28,29,31,33,35}

def spin():
    """Simulate a single spin of the roulette wheel."""
    return random.choice(pockets)

def get_color(result):
    """Determine the color of a roulette result."""
    if result in (0, "00"):
        return "green"
    elif result in red_numbers:
        return "red"
    else:
        return "black"

# Run 10 spins
for i in range(10):
    result = spin()
    color = get_color(result)
    print(f"Spin {i+1}: {result} ({color})")

3. Expected Value Calculator

Build a function that calculates EV for any roulette bet. Demonstrates functions, dictionaries, and mathematical thinking.

expected_value.py 
def expected_value(win_payout, win_prob, bet_amount=1):
    """
    Calculate the expected value of a roulette bet.
    
    Args:
        win_payout: Amount won (not including original bet back)
        win_prob: Probability of winning (0-1)
        bet_amount: Amount wagered
    
    Returns:
        Expected value per bet
    """
    lose_prob = 1 - win_prob
    ev = (win_payout * win_prob) + (-bet_amount * lose_prob)
    return ev

# Calculate EV for all standard US roulette bets
bets = [
    ("Straight Up",  35, 1/38),
    ("Split",        17, 2/38),
    ("Street",       11, 3/38),
    ("Corner",        8, 4/38),
    ("Six Line",      5, 6/38),
    ("Dozen/Column",  2, 12/38),
    ("Red/Black",     1, 18/38),
    ("Top Line",      6, 5/38),  # worst bet!
]

print("Bet              Payout   Win Prob   EV per $1    House Edge")
print("─" * 62)
for name, payout, prob in bets:
    ev = expected_value(payout, prob)
    house_edge = -ev * 100
    print(f"{name:17} {payout:5d}:1   {prob:8.4f}   ${ev:+.4f}      {house_edge:.2f}%")

# How much do you expect to lose over an evening?
bets_per_hour = 40
hours_playing = 3
bet_size = 10  # dollars
total_bets = bets_per_hour * hours_playing
house_edge_pct = 5.26 / 100

expected_loss = total_bets * bet_size * house_edge_pct
print(f"\n💸 Playing {hours_playing}hrs at ${bet_size}/bet, {bets_per_hour} bets/hr:")
print(f"   Expected loss: ${expected_loss:.2f}")

4. Monte Carlo Simulation with Visualization

Run thousands of simulated games and visualize the results using matplotlib. This is the gold standard technique for understanding probabilistic systems.

monte_carlo_roulette.py 
import random
import matplotlib.pyplot as plt
import numpy as np

def simulate_session(n_spins, bet_type="red", bet_size=10):
    """Simulate a roulette session and track bankroll over time."""
    pockets = [0, "00"] + list(range(1, 37))
    red = {1,3,5,7,9,12,14,16,18,19,21,23,25,27,30,32,34,36}
    
    bankroll = 0  # track cumulative profit/loss
    history = [0]
    
    for _ in range(n_spins):
        result = random.choice(pockets)
        if bet_type == "red":
            if result in red:
                bankroll += bet_size  # win 1:1
            else:
                bankroll -= bet_size
        history.append(bankroll)
    
    return history

# Run multiple sessions to see the spread of outcomes
n_sessions = 100
n_spins = 500

plt.figure(figsize=(14, 8))
plt.style.use('dark_background')

all_final = []
for i in range(n_sessions):
    history = simulate_session(n_spins)
    all_final.append(history[-1])
    alpha = 0.15 if n_sessions > 20 else 0.5
    color = '#15c96b' if history[-1] >= 0 else '#e23636'
    plt.plot(history, alpha=alpha, linewidth=0.8, color=color)

# Expected value line
ev_per_spin = -10 * (2/38)  # EV for $10 red bet
ev_line = [ev_per_spin * i for i in range(n_spins + 1)]
plt.plot(ev_line, color='#d4a843', linewidth=2.5, label='Expected Value')

plt.axhline(y=0, color='white', linewidth=0.5, alpha=0.3)
plt.xlabel('Spin Number')
plt.ylabel('Profit / Loss ($)')
plt.title(f'Monte Carlo: {n_sessions} Sessions × {n_spins} Spins ($10 Red Bets)')
plt.legend()

# Stats
winners = sum(1 for f in all_final if f > 0)
print(f"Sessions profitable:  {winners}/{n_sessions} ({winners/n_sessions*100:.1f}%)")
print(f"Average final P/L:   ${np.mean(all_final):.2f}")
print(f"Std deviation:       ${np.std(all_final):.2f}")
print(f"Best session:        ${max(all_final):.2f}")
print(f"Worst session:       ${min(all_final):.2f}")

plt.tight_layout()
plt.savefig('monte_carlo_roulette.png', dpi=150)
plt.show()

5. Convergence & Law of Large Numbers

Watch how the observed red frequency converges to the theoretical value as sample size grows. A powerful visual proof of the Law of Large Numbers.

convergence.py 
import random
import matplotlib.pyplot as plt

pockets = [0, "00"] + list(range(1, 37))
red = {1,3,5,7,9,12,14,16,18,19,21,23,25,27,30,32,34,36}

N = 10_000
theoretical = 18 / 38

# Track running frequency
red_count = 0
frequencies = []

for i in range(1, N + 1):
    result = random.choice(pockets)
    if result in red:
        red_count += 1
    frequencies.append(red_count / i)

plt.figure(figsize=(12, 6))
plt.style.use('dark_background')
plt.plot(frequencies, color='#e23636', alpha=0.8, linewidth=0.8)
plt.axhline(y=theoretical, color='#d4a843', linestyle='--',
            linewidth=2, label=f'Theoretical: {theoretical:.4f}')
plt.xlabel('Number of Spins')
plt.ylabel('Observed Red Frequency')
plt.title('Law of Large Numbers: Red Frequency Convergence')
plt.legend()
plt.tight_layout()
plt.savefig('convergence.png', dpi=150)
plt.show()
